# Learning log

This is an engineering log, not a diary. Add one entry at each milestone gate
and whenever a failure teaches us something important.

## Entry template

### YYYY-MM-DD - Topic

- Prediction: What did I expect to happen?
- Observation: What actually happened?
- Explanation: Why did the system behave that way?
- Change: What did I change, if anything?
- Prevention: Which test, check, metric, or design rule prevents regression?
- Open question: What remains uncertain?
